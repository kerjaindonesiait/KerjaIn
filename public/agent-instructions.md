# KerjaIn — Agent Runbook

KerjaIn adalah marketplace tukang rumah di Jakarta, Indonesia.

## Scope

- Gunakan hanya halaman publik yang tercantum di `/llms.txt`.
- Jangan mengakses area login, dasbor, atau pesan tanpa persetujuan pengguna.
- API backend (`api.kerjaindonesia.com`) memerlukan autentikasi; jangan memanggil endpoint privat.
